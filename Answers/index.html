Задание №1: Дан код:
var a = 1, b = 1, c, d;
c = ++a; alert(c); // 2
d = b++; alert(d); // 1
c = (2+ ++a); alert(c); // 5
d = (2+ b++); alert(d); // 4
alert(a); // 3
alert(b); // 3
 Почему код даёт именно такие результаты?

1) ++a операция инкремента, a = 1 + 1 = 2.
2) b++ постинкремент - сначала переменной присваивается значение, а потом значение b увеличивается на 1.
3) Прежде переменная a стала равна 2. Теперь инкрементируется еще и равна 3. 2 + 3 = 5.
4) После предыдущего постинкремента b = 2. 2 + 2 = 4. Далее переменная b увеличивается еще на 1.
5) Из прошлых пунктов ясно, что a = 3 и b после постинкремента стала равна 3.
